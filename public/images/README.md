# Archivo para mantener la estructura del directorio

# Las imágenes de productos se pueden guardar aquí

## Recomendaciones:

- Usa imágenes cuadradas (1:1) para mejor visualización
- Tamaño recomendado: 500x500 píxeles
- Formatos: JPG, PNG, WebP
- Peso máximo: 5MB por imagen

## Ejemplo de estructura:

```
images/
├── productos/
│   ├── manzana.jpg
│   ├── leche.jpg
│   └── pan.jpg
└── categorias/
    ├── frutas.jpg
    └── lacteos.jpg
```

## Fuentes de imágenes gratuitas:

- Unsplash: https://unsplash.com/
- Pexels: https://www.pexels.com/
- Pixabay: https://pixabay.com/
